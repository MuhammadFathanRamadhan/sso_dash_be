# sso_dash_be
REST API backend for BOMA SSO Dashboard built with Go, Fiber, GORM, and MySQL. Features passwordless OTP-based authentication, JWT session management with multi-device tracking, login alerts, connected app management, account deletion flow, and IP-based rate limiting. Includes auto-generated Swagger documentation.
